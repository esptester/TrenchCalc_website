QUICK SETUP: Admin Cognito User Pool
=====================================

STEP 1: Create Admin User Pool (AWS Console)
---------------------------------------------
1. AWS Console > Cognito > User Pools > Create user pool
2. Sign-in: Email only
3. MFA: Required (TOTP only)
4. Sign-up: Disable (manual only)
5. Name: "engsite-admin-pool"
6. Create

STEP 2: Create App Client
---------------------------
1. User Pool > App integration > Create app client
2. Name: "WebAdminClient"
3. Client secret: ❌ UNCHECK (no secret!)
4. Auth flows: ✅ ALLOW_USER_PASSWORD_AUTH
5. Auth flows: ✅ ALLOW_REFRESH_TOKEN_AUTH
6. Create

STEP 3: Get IDs
---------------
1. User Pool ID: ap-southeast-2_XXXXXXXXX (General settings)
2. Client ID: 1a2b3c... (App clients)
3. Copy both

STEP 4: Update cognito-config.js
---------------------------------
1. Open cognito-config.js
2. Paste User Pool ID
3. Paste Client ID
4. Save

STEP 5: Create Admin Users
---------------------------
1. User Pool > Users > Create user
2. Email: admin@engsitetools.com
3. Email verified: ✅ Checked
4. Temporary password: Send invitation
5. Create

STEP 6: User Setup (First Login)
---------------------------------
1. User logs in with temp password
2. Must change password
3. Must set up MFA (TOTP - Google Authenticator)
4. Done!

COST: $0 (free tier: 50k MAU)
SECURITY: High (separate pool, MFA required)
